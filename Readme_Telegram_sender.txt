Инструкция по настройке плагина

Шаг 1: Создание Telegram бота
    Найти в Telegram бота @BotFather
    Отправить команду /newbot
    Ввести имя бота (например: MyFileSenderBot)
    Получить токен бота (сохранить его)

Шаг 2: Получение ID канала
    Добавить бота в канал как администратора
    Отправить в канал любое сообщение
    Перейти по ссылке: https://api.telegram.org/bot<ВАШ_ТОКЕН>/getUpdates
    Найти chat.id в ответе (для каналов обычно начинается с -100)

Шаг 3: Настройка плагина в программ
    Запустить программу
    Перейти на вкладку "Telegram Отправка"
    Ввести полученные данные:
        Токен бота: токен из @BotFather
        ID канала: ID канала (например: -1001234567890)
        Задержка: время в секундах перед отправкой
        Папка мониторинга: папка для отслеживания файлов
        Расширения: типы файлов для отправки
    Нажать "Сохранить настройки"
    Нажать "Запуск мониторинга"

Особенности плагина:
    ✅ Настраиваемая задержка - файлы отправляются не сразу
    ✅ Отслеживание папки - автоматическое обнаружение новых файлов
    ✅ Фильтрация по расширениям - отправляются только нужные типы файлов
    ✅ Логирование - подробный лог всех операций
    ✅ Тестовая отправка - проверка настроек перед запуском
    ✅ Безопасная работа - обработка ошибок и исключений

Плагин готов к использованию! После настройки он будет автоматически отправлять файлы в Telegram канал с указанной задержкой.
